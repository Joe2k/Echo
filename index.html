Echo

Music Bot For Discord

Features

Command Meaning
>play [name or link]
Play any song
>playlist [playlist link]
Imposts a playlist to the queue
>search [name]
Displays the top 5 search results for that name from which you can add to queue
>stop
Stop the queue and leave
>pause
 Pause the queue
>resume
 Resume the queue
>skip
 Skip to the next song in the queue
>queue
Display the queue
>np
 Display the current song
>remove [number]
 Removes that song with that number from the queue
>help Displays the help list
